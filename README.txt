This simple online character is easy to run. you can find the live version of it at http://ene2.mygamesonline.org/ecalc/

To use it first register an account using full name, username and password.
Then login to the web app using your credential.
After login the first landing page is the place where you can compute your expression with just a single click.
Then there is a history page which you can use and customize it as your wish to show the range of data you requested.
The last page you can find is a profile page where you can manage your profile either as a whole, solely, or multiple at the same time.
